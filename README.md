# Система учета и анализа посещаемости СК "Салют"

Проект выполнен по теме:
`Проектирование и разработка системы учета и анализа посещаемости спортивного клуба "Салют"`

## Технологии
- C# / .NET 9 (ASP.NET Core MVC)
- SQLite (EF Core)
- HTML/CSS
- Bootstrap
- Chart.js

## Ключевые функции
- Учет клиентов клуба (CRUD)
- Учет тренировок и расписания (CRUD)
- Регистрация посещений (CRUD)
- Бизнес-валидация регистрации посещений:
  - проверка существования клиента/тренировки
  - проверка срока действия абонемента
  - проверка вместимости тренировки
  - проверка допустимого диапазона даты посещения
- Аналитическая панель:
  - посещения за 14 дней
  - структура посещений по типам абонементов
  - топ тренировок за 30 дней
- Обработка ошибок в контроллерах с логированием и пользовательскими уведомлениями
- Экспорт журнала посещений в CSV по выбранным фильтрам

## Быстрый запуск
```bash
dotnet restore

dotnet run --project SalutClubAttendance.Web
```

Открыть: `http://localhost:5078` (или адрес из вывода консоли).

## Тесты
```bash
dotnet test SalutClubAttendance.sln
```

Покрыты сценарии:
- корректная сборка ключевых метрик аналитики;
- поведение аналитики при отсутствии посещений;
- ограничение выдачи блока топ-тренировок;
- валидация регистрации посещения (валидный случай и основные ошибки).

## Структура
- `SalutClubAttendance.Web` - основное веб-приложение
- `SalutClubAttendance.Tests` - тестовые сценарии (xUnit)
- `SalutClubAttendance.sln` - файл решения

## Примечание по данным
При первом запуске БД SQLite (`salut_club_attendance.db`) создается автоматически и заполняется демонстрационными данными через `DbInitializer`.
